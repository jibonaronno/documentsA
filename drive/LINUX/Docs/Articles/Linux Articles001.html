Prepare your disk for embedded computer.

Two primary partitions needed.
First one is bootable and contains Kernel, some Booting Files, initrd, some data files to put outside of initrd if needed. So if the embedded app needed to access data from first pertition, then it is needed to mount the first pertition. The first pertition will be mounted as read-only. So the boot pertition will be intact in case of power failure.
The second pertition is for Read-Write data files used by the embedded application. Initially mount as read only, then before write data remount as read-write permission, after writing the data remount again as read only.
