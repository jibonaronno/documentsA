LU FPGA DSP AND ARM LINUX


Hand Drawing diagram of LU IED structure. FPGA collects data from 4/5 Analog Devices 24Bit ADC and send it to DSP over PCIe bus. The DSP and the Sitara processor are in one package. The DSP runs a FreeRTOS app with 5/6 tasks and shares a memory address or memory area as a DTB device accessible to Linux system.
