pdtester.py

Open this code in PyCharm editor. Use Pycharm Community version. You can use any Python version greater than 3.9 . pdtester.py file is in ai-server folder. You can install the modules from pycharm by hovering the mouse pointer on the module name and click install module. You also need to install cuda if you want to train new model. The raw binary files are in sanion_python_projects\wavegenA_PD_FILTER_GRAPH\PD_TESLA_COPY_A location. Load this folder to see the list of files, then click any file to see the data. Then click apply kalman to see kalman output. Click on LSTM to see LSTM output. The LSTM output does not look right. LSTM OUTPUT should match with the Kalman Output. You have to train LSTM for better output.
